# Student Management System

A simple console-based Java project built using OOP concepts and the Java Collections Framework.

This project allows users to manage student records through a menu-driven program.

---

# Features

- Add Student
- Display All Students
- Search Student by ID
- Remove Student
- Update Student Details
- Sort Students by Name
- Sort Students by Marks
- Menu-driven console application

---

# Concepts Used

## OOP Concepts
- Classes & Objects
- Encapsulation
- Constructors
- Method Overriding

## Java Collections Framework
- ArrayList
- Iterator
- Collections.sort()
- Comparator

## Other Concepts
- Scanner Class
- CRUD Operations
- Menu-driven Programming
